
token 模块

接收 字符串
返回 参数数组 (参数1 是函数名)

parse 模块

接收 参数数组
返回 表达式链表

eval 模块 (或者其他名字?)
接收 表达式链表
     依次执行表达式
返回 表达式的结果

memory 模块
维持一个栈
维持一个堆
提供申请内存接口
提供在表达式中的变量都放在栈上, 依次累计, 表达式计算完之后释放.

function 模块
维持一个 function 列表

var 模块
接收 字符串
返回 一个var结构体


表达式模块

表达式结构体:
    通过参数数组上次呢过
    组织成: 函数 (参数列表) 的形式

表达式执行:
    找到第一个函数, XX(并将一次传入参数的值(如果参数是一个表达式则先执行))
    	and pass the args into the function,
    	the value of params will be calculate by function
